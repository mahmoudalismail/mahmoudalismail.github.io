Research

Enhancing Agent Gaze In Uncontrolled Environments
Funded and Sponsered by QSIURP

Utilized and enhanced state of the art face detection algorithms to direct the gaze of an agent (Hala, bilingual Robo-recptionist) towards an interlocutor in uncontrolled environments using Microsoft Kinect's sensor.
Hala was able to locate the interlocutors face in severe daylight changes with different accuracies under three locations. However, the experiment we conducted shows that identifying faces in our uncontrolled environment poses difficulties and needs an alternative approach.
